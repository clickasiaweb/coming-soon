# Alcant Coming Soon Page

A professional coming soon page for Alcant website, designed for Vercel deployment.

## Files Structure

```
alcant-coming-soon/
├── index.html    # Main HTML structure
├── style.css     # Modern dark theme styling
├── vercel.json   # Vercel deployment configuration
└── README.md     # This file
```

## Deployment Instructions

### Option 1: Using Vercel CLI

1. Install Vercel CLI:
   ```bash
   npm i -g vercel
   ```

2. Navigate to project directory:
   ```bash
   cd "d:\AEROSPACE ENGINEERING\Web\WEBSITE\Comingsoon"
   ```

3. Deploy to Vercel:
   ```bash
   vercel
   ```

4. Follow the prompts to configure your project and custom domain.

### Option 2: Using Vercel Dashboard

1. Push this repository to GitHub/GitLab/Bitbucket
2. Connect your repository to Vercel
3. Vercel will automatically detect and deploy the project
4. Configure your custom domain in the Vercel dashboard
